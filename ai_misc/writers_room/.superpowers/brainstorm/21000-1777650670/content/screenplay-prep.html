Project Prep Status

project.prep_status tracks where a project is in the screenplay prep workflow:

concept → seed_growing → visualization
→ research → references → identity → ready

Only projects in ready status can have scenes created and run. Writers see a prep checklist dashboard per project.

Completes & Exports

At any point a Writer/Producer can export:
→ Concept Document — story summary, arcs, conflict plan
→ Character Profiles — full character + arc sheets
→ Research Plan — world-building and fact needs
→ Reference Library — similar works and inspiration
→ Project Identity Package — title, logline, synopsis
→ Full YAML export — entire project for portability
